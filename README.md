# EliteTips Solo – Complete Prediction Platform

A complete, ready-to-deploy prediction platform with M-PESA and PayPal integration.

## Features

- ✅ Admin dashboard to create and manage predictions
- ✅ Client dashboard to view predictions after payment
- ✅ M-PESA integration (IntaSend) for Kenyan clients
- ✅ PayPal integration for international clients
- ✅ Paystack integration for card payments
- ✅ 24-hour access granted after payment
- ✅ Full Next.js 15 source code
- ✅ PostgreSQL database (Prisma ORM)
- ✅ Responsive dark theme UI

## Quick Install

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up PostgreSQL database
4. Copy `.env.example` to `.env` and fill in your details
5. Set up your payment accounts (IntaSend, PayPal, Paystack)
6. Run migrations: `npx prisma migrate deploy`
7. Build and start: `npm run build && npm start`
